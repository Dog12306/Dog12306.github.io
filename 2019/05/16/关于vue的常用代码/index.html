vue

渐进式 JavaScript 框架

vue 基础语法

插值

```{{}}```
  • 直接插值

指令

v-xx开头的都是指令

  • v-html 用于直接展示 html解析后的结果
  • v-bind 可用把某个属性修改为动态 直接调用数据对象的值
  • v-if 是否渲染 ,跟display无关
  • v-on 给元素绑定事件
  • v-bind 缩写 :
  • v-on 缩写 @

计算属性

computed 一般用于对数据对象进行修饰,用于页面展示.计算属性是依赖型的,只要依赖的数据对象没有发生变化,就不会重新计算.

方法

methods 一般用于内部调用,每次调用都会重新计算.

watch

监听作用于属性变化

lodash

  • debounce 防抖
  • throttle 限流

样式

  • class

    • 对象形式
    • 数组形式
    • 三元表达式

  • style

    • 对象形式
    • 数组形式
